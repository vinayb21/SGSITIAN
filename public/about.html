About Us

Introduction

SGSITIAN is aimed for students and faculty members of SGSITS, Information Technology (IT) department. Creating a platform where students and faculty members share information regarding their curriculum, internship and job opportunities.


Technologies Used

SGSITIAN is a SPA (Single Page Application), built on top of MEAN Stack.
The components of the MEAN stack are as follows -

  • MongoDB, a NoSQL database
  • Express.js, a web application framework that runs on Node.js
  • Angular.js, a JavaScript MVC framework that run in browser JavaScript engines
  • Node.js, an execution environment for event-driven server-side and networking applications

AJAX (Asynchronous Javascript and XML) used to reload the required content without reloading the whole web-page.

Representational state transfer (REST) or RESTful Web services used for providing interoperability using a uniform and predefined set of stateless operations.

Developed with mobile-first approach, so easily accessible on all devices.

jQuery (javascript library) used to make components more interactive.

Bootstrap (Front-end framework) used to make a sleek, intuitive, and powerful mobile first front-end for faster and easier web development.

HTML5, CSS3 & Javascript for structuring, styling and making the website dynamic.



Scope

Easily accessible, fast and light-weight as data is saved by asynchronous reloading of pages. Design being simple yet effective attracts the attention of users. Can be integrated with further features like chat services for group chat, peer to peer chat etc.
